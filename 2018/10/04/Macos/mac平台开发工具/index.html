我用的mac平台开发工具

轻量级多语言代码编辑器

这些编辑器资源占用小、打开速度快，适合想省电或是代码量小时使用。

  • coderunner 2

我小型程序一般使用这个，优点是上手非常容易还可以直接运行和调试，并且把常用的功能都放在了Touch Bar上，可以非常方便地运行、停止、注释、缩进，而且是mac本土软件。

  • sublime 3

优点非常多，必备软件。

有专门的包管理软件，直接搜索下载就可以安装任何冷门语言包。

能修改查看文件使用的编码。

重量级的代码编辑器

apple开发

xcode

c/c++

clion

java

idea

python

pycharm

html/css

dreamweave

数据库可视化

DataGrip （jetbrain出品，必属精品）

Navicat （感觉操作很麻烦，很多操作只能点击，没有快捷键，而且程序奔溃多次，个人不推荐）

PostgreSQL 的可视化：Postico （非常清爽便捷，十分喜欢这个）

Redis 的可视化：Medis

虚拟机

virtual box （免费）

VMware Fusion（非常好用）

远程桌面

remotix （支持各种连接，我主要用来连接windows）

mac有自带的vnc（ finder 内按 command+k ），所以可以直接连Linux。

离线文档手册

Dash （mac端的福利，可以离线各种文档，并且可以在jetbrain等第三方软件内设置dash查阅的快捷键）

markdown编辑器

typora，因为window上用习惯了，mac上表现也很好。免费。
