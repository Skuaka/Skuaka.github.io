交换

VLAN

企业级

  • 原理层面：vlan 用于分隔广播域
  • 部署层面：一个部门一个 vlan
  • 配置层面：
  • 排错层面：忘记创建；忘记把接口加入、加入错。

Trunk-link

企业级

  • 原理层面：承载携带不同 vlan 标记的流量；vlan标记的携带方式：DOT1Q、ISL。
  • 部署层面：基础配置、修改本帧vlan、修改Trunk-link vlan承载列表。
  • 配置层面：
  • 排错层面：封装标准不一致；错误的承载列表
  • vlan间通讯原则

RSTP

企业级

  • 原理层面：

    STP、RSTP 计算出无环的交换路径，并且保证一定的冗余性。

    根桥的竞选，根端口的竞选，指定端口的竞选，

    根桥如何故障恢复，RSTP 和 STP 的区别。

  • 部署层面：如何影响根及根端口选举结果；PVST（每vlan生成树）

  • 配置层面：

  • 排错层面：确认根桥、端口是否正确

MSTP

园区网络、运营商级

  • 原理层面：

    允许多个vlan映射到一个实例中，并且层次化、区块化维护生成树。

    CST（通用生成树）、IST（内部生成树）

  • 部署层面：IST的规划、CST的定义

  • 配置层面：

  • 排错层面：MSTP 的 IST 的域名不一致，口令不一致

EC：以太网通道

企业、园区、运营商级

  • 原理层面：EC 允许交换机把最多8个物理接口捆绑成一个逻辑接口。
  • 部署层面：
  • 配置层面：
  • 排错层面：端口没有统一绑定到一个逻辑接口，接口的物理参数和逻辑参数不一致。

  • 部署层面：

    HSRP 注意 active 和 standby 分配；

    VRRP 注意 master 和 backup 的分配；

    GLBP 注意权重分配。

路由

OSPF

企业、园区、运营商级

  • 原理层面：

    OSPF 邻接关系建立条件和规则；

    OSPF DR 和 BDR 的选举；

    OSPF 的开销计算；

    OSPF LSA 的类型、作用、扩散范围、对应的路由；

    OSPF 的孤岛区域、区域分割、虚链路；

    OSPF 的区域汇总；

    OSPF 的特殊区域；

    OSPF 的区域认证、链路认证；

  • 部署层面：不要出现孤岛、区域分割问题；小心特殊区域的优化。

    P-NETWORK、P、PE、C-NETWORK、CE、VRF、RD、RT、顶部标签、底部标签、数据传递过程。
