安装

yum install postgresql postgresql-server

远程连接配置

修改/var/lib/pgsql/data/postgresql.conf

listen_addresses = '*' //监听所有ip的连接，默认是本机

修改/var/lib/pgsql/data/pg_hba.conf

host all all 0.0.0.0/0 md5 //添加这行，允许所有IP登入

openssl加密(可选)

转自：https://my.oschina.net/tianbing/blog/744826

  1. 利用openssl生成私钥和证书，可以写成一个shell脚本来实现。

在执行脚本文件前，注意定义好data目录的环境变量。执行脚本后就会发现在data目录下生成两个文件：私钥server.key和证书server.crt。在shell脚本中需要对这两个密钥修改一下权限，否则数据库起动不了。

  1. 修改postgresql.conf配置文件，打开ssl连接。
  1. 修改pg_hba.conf配置文件，以hostssl方式连接。
