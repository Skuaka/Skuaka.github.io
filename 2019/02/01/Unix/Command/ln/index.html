ln

ln -s 源文件 目标文件

举例：

当前目录是/local，而我经常要访问/usr/local/linux/work

那么我就可以使用在 local 下建立一个文件 linkwork，

然后ln -s /usr/local/linux/work /local/linkwork

即建立两者之间的链接。

删除链接

rm -rf symbolic_name 注意不是 rm -rf symbolic_name/

那么上面就是

rm -rf /local/linkwork
