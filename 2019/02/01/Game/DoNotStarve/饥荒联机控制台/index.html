饥荒联机控制台

当你为服务器管理员时，在游戏中按 ~ 呼出控制台，再按 Ctrl 键即可转换为 Remote 模式，会出现 “Remote” 字样。

你在重新加载世界后必须重新输入每一个命令。

简单命令

#####生成实例

在鼠标所指处生成一定数量被指定的”预设”。

#####获得物品

在你的物品栏中生成一定数量被指定的”预设”。只适用于可以被储存在物品栏的背包和物品 。

#####生命值

用百分比设置你的生命值。注：使用分数 0.90 = 90%

#####理智值

用百分比设置你的理智值。注：使用分数 0.90 = 90%

#####饥饿值

用百分比设置你的饥饿值。注：使用分数 0.90 = 90%

#####湿度

用百分比设置你的湿度。注：使用分数 0.90 = 90%

#####温度

用百分比设置你的温度。注：使用分数 0.90 = 90%

#####-上帝模式

玩家的理智值、饥饿值和被攻击时生命值不再下降。上帝模式下，你不能用命令改变的你统计数据。如果处于死亡状态，将会复活你（通过 c_godmode() 复活后不会将你置于上帝模式）。再次使用命令停用上帝模式。

#####超级上帝模式

和上帝模式一样但会将你的各项统计数据设置为满，即使你将它关闭。与c_godmode()相似,，它也会复活你（通过 c_supergodmode() 复活后不会将你置于上帝模式）。再次使用命令停用上帝模式。

#####-移动速度

标准额外的移动速度为 1。设置为 2 使你以两倍的速度移动，设置为 20 可以让你轻松地穿过墙和 “越过 “ 水。

玩家命令

#####-创造模式

你可以合成任何东西。

#####生命值上限

改变你的角色 的生命 值上限

#####理智值上限

改变你的角色 的理智 值上限

#####饥饿值上限

改变你的角色 的饥饿 值上限

#####暂停饥饿

你的角色 不再降低饥饿值。

#####-伤害倍数

改变你的角色 的伤害倍数

#####海狸

把伍迪 变成海狸 。把 .01 改为 1 可以让你变回伍迪。

其他玩家命令

#####标明一个玩家

AllPlayers[1] 取的是主机玩家。其他玩家的编号显示在记分板上（某些情况下编号有可能是错的，你可以先用 c_listallplayers() 来查看每个用户编号的用户名和角色）。大部分玩家命令可以用 AllPlayers[number] 代替 ThePlayer。

-对他人执行命令

先使用 c_select(AllPlayers[number])可以选定他人，然后执行命令。

#####对所有玩家使用同一命令

只修改command，用其他命令代替command，使用 “v” 代替命令中的 AllPlayers[number]。例如， “for k,v in pairs(AllPlayers) do c_move(v) end” 将会把所有玩家都移动到鼠标所指的位置。

#####-移动玩家

移动玩家到鼠标所指的位置。

#####杀死玩家

杀死玩家。

#####-复活玩家

复活玩家。

#####-传送至玩家

传送到对应编号的玩家，玩家编号可以通过 c_listallplayers() 获取。

#####丢掉物品栏物品

丢掉对应编号的玩家物品栏的所有物品，玩家编号可以通过 c_listallplayers() 获取。

#####-重新选择角色

使用该命令将会删除玩家物品栏的物品而不会掉落，所以推荐你先让玩家的物品丢掉他们的物品，或者使用上一条命令 DropEverthing()。

世界命令

#####传送到预设

输入该命令后，你将被传送到第一个预设的实体.如果存在多个预设的实体，控制台日志将会显示实体数量的列表，再次执行相同的命令将会传送玩家到下一个在世界中生成的预设的实体。

#####删除鼠标下的实体

输入该命令后，将会删除你鼠标下的物品。在专用服务器上使用第二个命令，因为第一个不能在专用服务器上使用。

#####跳过一天

#####跳过时间

跳过 x 天。将 x 修改为所要跳的天数（例如，16\30*4.5 为跳过4.5天） 警告：*数值太大可能会冻结游戏。（取决于电脑的运算速度）

#####-跳过时钟阶段

#####设置季节长度

设置季节的长度。

#####换季

#####开始下雨

#####停止下雨

停止下雨。这包括青蛙雨 。

#####生成闪电

闪电击中玩家，如果靠近避雷针的话，则击中避雷针。

#####陨星撞击

陨星撞击在鼠标的位置。随机生成不同类型的岩石。

网络/服务器命令

#####重新加载世界

不保存游戏并重载世界（也就是重载到上一个储存点）。如果你是一个客户端的话，该命令可能会造成你的游戏崩溃（除非你用远程命令模式发送该命令）。

#####重新生成世界

#####保存

强制服务器立即保存。（服务器通常在夜晚结束的时候自动保存）

#####关闭服务器

true 会保存游戏，false 不会保存游戏。 c_shutdown() 和 c_shutdown(true) 是一样的。

#####-回档

回档服务器一定次数。 c_rollback() 会回档一次，c_rollback(3) 会回档三次。

#####发送公告

可以发送一个公告给玩家，例如即将关闭 / 重启服务器，让玩家知道即将断线。

其他

生成虫洞
  1. 没有只用一行的命令，请逐步输入
  2. 用这个命令生成虫洞1： worm1 = c_spawn(“wormhole”)
  3. 用这个命令生成虫洞2： worm2 = c_spawn(“wormhole”)
  4. 创建虫洞1到虫洞2之间的连接： worm1.components.teleporter.targetTeleporter = worm2
  5. 还有回来的连接： worm2.components.teleporter.targetTeleporter = worm1
生成一只被驯化了的牛

用下面的值替换 DEFAULT 来改变牛的特征。”RIDER”, “ORNERY”, “PUDGY”

-修改角色大小

附

Spawning Items

There are two approaches in spawn items:

  • Using c_spawn in Console

It spawns an item or a mob at the position of mouse click immediately.

  • Using c_give in Console

    It directly gives an item to the player’s inventory immediately.
