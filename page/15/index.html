Html入门

html(hyper markup language)超文本标记语言，是一种标准—–w3c

超文本：在文本内容的基础上，添加更加丰富的信息（图片，动画，声音等）

html基本骨架结构

html结构说明

  • <html>含义：告诉浏览器网页的代码用什么格式来解析

  • <head>含义：告诉浏览器网页的文字用什么字符集，字符集不正确则会出现乱码

    gb2312 简体中文 —- 汉字操作系统默认使用的字符集

    gbk 国际码 utf-8 多国语言

  • <title>含义：便于搜索引擎查找

  • <body>含义：网站的内容只有放入该标签才能在窗口中显示

html书写规范

  • 不区分大小写，但是w3c使用的是小写
  • 单标签一定要闭合<br />
  • 属性值用双引号

html文本标签

  • <font color="red" size="5" face="楷书">content</font>size以号为单位
    1. <b></b>加粗
    2. <strong></strong>加粗 加强语气
    3. <i></i>倾斜
    4. <em></em>倾斜 加强语气
    5. <u></u>下划线
    6. <ins></ins>下划线 强调内容
    7. <sup></sup>上标
    8. <sub></sub>下标
  • 段落标签<p></p>属性：Align 取值 left center right
  • 标题标签<h1></h1>...<h6></h6>属性：Align
  • pre 预定义标签<pre>content</pre>保留标签之间内容的空格和换行
  • div 本身没有意义，自己独占一行，结合css 块标签
  • span 本身没有意义，结合css 行内标签

html图片标签

语法：<img 属性="属性值"/>

属性：

  • 路径 src=”图片地址“ 图片要放到同一个站点（文件夹）下

  • 宽度 width=“1” 以像素px为单位，不用写单位

  • 高度 weight=“1”

  • 边框 border=“1”

  • 解释说明 alt=“对图片的描述内容”

  • 图片和内容左右之间的距离 hspace=“1” （css实现更佳）

  • 图片和内容上下之间的距离 vspace=“1” （css实现更佳）

    注意：只设置高度或宽度，图片等比例缩放

html列表

  1. 无序列表

    属性：type 设置列表前面符号的样式 取值 circle(实心圆) disc(空心圆 默认) square(方形)

  2. 有序列表

    <ol>类似

html表格

  • table属性：

    1. 边框 border=“2” 默认是0

    2. 宽度 width=“2”

    3. 高度 height=“2”（w3c不建议使用高度）

    4. 居中方式 align=“center”

    5. 表格中内容和单元格之间的距离 cellpadding=“2” 默认是2px

    6. 表格中单元格和单元格之间的距离 cellspacing=“2” 默认是2px

    7. 背景颜色 bgcolor=“red”

    8. 背景图片 backgroun=“图片地址” (不能含有中文)

      注意：背景图片的优先级高于背景颜色

    9. 合并边框线 rules=“all”（css实现更佳）

    10. 边框颜色 bordercolor=“red”（css实现更佳）

  • tr 行属性

    1. 高度 height=“300”
    2. 背景颜色 bgcolor=“blue”
    3. 背景图片 background=“图片地址”
    4. 水平对齐方式 align=“left//center//right”
    5. 垂直对齐方式 valign=“top//middle//bottom”

  • td 单元格属性

    1. 背景颜色 bgcolor

    2. 背景图片 background=“图片地址”

    3. 宽度 width=“300”

    4. 高度 height=“40”

    5. 水平对齐方式 align=“left//center//right”

    6. 垂直对齐方式 valign=“top//middle//bottom”

    7. 合并单元格

      横向合并 colspan=“2” 横向合并两个

      纵向合并 rowspan=“3”

html链接

语法：<a 属性=“属性值”>content</a>

属性：

  • 链接的地址 href=“地址 url” 相对地址或绝对地址

    2. 默认的打开方式，在原窗口打开目标文件 _self
  • name 定义锚点的名称

  • 邮件连接

    <a href="mailto:邮件地址">content</a>

  • 空链接

    链接到当前页面 <a href="#">content</a>

  • javascript 链接

    <a href="javascript:window.close()">关闭页面</a>

    Firefox设置问题：1在地址栏输入about config 2在搜索中输入close_windows 3双击配置改为true

锚点：用name属性定义锚点名称

定义锚点<a name="anchor"></a>标签之间不加内容

跳转锚点<a href="#anchor"content</a>

Meta标签

描述网页文档属性

http-equiv    和    name

http-equiv 模拟http文件头信息，当内容从服务器端发送客户端，告诉浏览器如何正确显示信息

  • 字符集

    <meta http-equiv="content-type"content="text/html;charset="UTF-8"/>多国语言

    字符集介绍

    • ASCII 2^8=256

    • ANSI 各个国家对ASCII编码进行扩展，用于显示本国的语言 2^16=65536

      中文操作系统 gb2312 收录了6763个汉字

      繁体操作系统 big5

    • GBK 对gb2312扩充，收录冷门字 越2.1万汉字

    • Unicode 计划将世上所有字符统一编码，用4Byte(32bit)表示一个字符，造成极大空间浪费

    • UTF-8 多国语言编码，它会选择合适编码来翻译不同的字符

  • 网页自动刷新

    <meta http-equiv="refresh" content="4"/>间隔4秒网页自动刷新

    <meta http-equiv="refresh" content="8;http://www.baidu.com"/>等待8秒自动跳转到百度

  • Name可以设置网页关键字，描述信息等

    <meta name="description" content="网站的描述信息"/>便于搜索引擎需要

    <meta name="author" content="作者"/>网站的作者

html表单

可以获取客户端的信息（数据）。

对表单中的数据进行验证，如果不符合制定的要求，提示验证没有通过；如果符合相应的要求，把数据提交到服务器端，数据放入数据库中。

表单分为前台的制作页面，php对数据的处理。

表单的结构

语法

表单的属性

  1. Name=”username” 表单的识别名称

  2. Action=”xx.php” 对数据进行处理

    Action=”” 对表单中的数据处理的程序就在当前文档

  3. Method=”get/post” 把表单中的数据提交到服务器端的方法，传递数据的方法

    1. Get 方法 默认的一种传递数据的方法，通过地址来传递表单中的数据。

      • 不能传递敏感的数据，密码
      • 不能传递大量的数据，每次只能传递1024字节。
      • 不能上传附件

    2. Post 方法，不是通过地址栏传递数据，数据传给文件处理程序

      • 相对安全
      • 可以传递海量数据
      • 可以传递附件

Xhtml

可扩展的超文本标记语言—–取代 html，pc机 —–面向当今的移动端设备

Xhtml语法比html严格：

  • 输入的标签和属性必须是小写

  • 单标签后必须加/ 例如<br />

  • 属性必须用双引号

  • 必须有 dtd （document type definition）文档类型定义
    dtd 是一种验证机制，检验输入的xhtml 是否符合规范
    dtd 分类：

    • 过渡型 xhtml1-transitional 允许使用表现标签和属性<b> <i> <font color="red">等

    • 严格型 xhtml1-strict（不常用）不允许使用表现标签和属性，必须使用css样式来设置

    • 框架型 xhtml1-frameset（不常用）给框架用

    • 可以通过该网站测试 dtd 声明 http://www.w3.org

CSS入门

CSS简介

CSS(cascading style sheet) 层叠样式表

语法：选择器{属性名:属性值; 属性名:属性值; }

说明：

  • 一个css样式包括选择器和格式声明语句
  • 选择器就是选择给哪个html标签加样式
  • 格式声明语句要用{ }，包括属性名：属性值；
  • 属性名，w3c 已经定义好，直接使用
  • 属性值不用双引号
  • 属性值多以像素px为单位，通常情况必须加单位（html属性不加单位）

CSS样式表

如何在html结构中引入css样式

  1. 行内样式表：

    每一个标签都有style属性

    ​ 格式：<标签 style=“属性名:属性值; 属性名:属性值;” >内容</标签>

    只能一个一个设置，麻烦，临时测试用

  2. 内嵌样式表：

    格式：在head标签中输入 (在xhtml中必须写type属性)

  3. 外部样式表：

    格式：<link rel="stylesheet"href="外部样式表文件 xx.css"/>

  4. 导入样式表：

    管理css样式，大型网站需要

    格式: @import url(另外一个css样式)

    注意: @import 是css样式标签，必须放到css文件中样式表的最上端

CSS选择器

基本选择器

  • 标签选择器 body{} p{} div{}
  • 类选择器

    每一个标签都有一个class属性<label class="myclass"></label>

    类用 .来表示 例如.myclass{ } .page_header{ }

  • id选择器

    每一个标签都有一个id属性<label id="myid"></label>

    id用#来表示 例如#myid{ }

    注意：id只能唯一，通常id给JavaScript用，不是用来设置样式的，如果想设置样式，用类选择器

  • 通用选择器

    给所有的html标签加样式

    语法：*{ }

    ie6版本不支持

复合选择器

  • 多元素选择器

    多个标签共有的属性和属性值，放到一起

    语法：选择器1,选择器2,选择器3……{ }

  • 后代元素选择器

    给html的后代元素（不仅是直接子元素）加样式

    语法：选择器1 选择器2 选择器3{ }

    以空格隔开，表明选择器1里面的选择器2里面的选择器3

  • 子元素选择器

    给html的直接子元素加样式

    语法：选择器>选择器{ }表明某个元素里面的第一层元素

伪类

—-锚<a>(内容必须做好链接)

a:link{ }未访问的链接

a:visited{ }访问过的链接

a:honer{ }鼠标移动到链接上

a:active{ }单击鼠标左键的那一刻的样式

文本的属性

  • font-size 字体大小 :12px;
  • font-weight 是否加粗 :bold//normal;
  • font-style 是否倾斜 :italic//normal;
  • font-family 字体样式 :隶书; 默认为宋体
  • font-decoration 是否有线条 :underline//overline//line-through//none;
  • text-indent 首行缩进 :2em;
  • color 颜色 :red;
  • letter-spacing 字母间距 :2px;
  • word-spacing 单词间距 :2px;
  • text-align 对齐方式 :left//center//right;
  • line-height 行间距 :150%;

背景的属性

  • background-color 背景颜色

  • background-image 背景图片

  • background-repeat 背景图片是否平铺 :no-repeat//repeat-x(横向平铺)//repeat-y(纵向平铺)//repeat(默认，横向纵向都平铺)

  • background-attachment 背景附件，背景是否随着上方的内容一起滚动 :fixed//scroll

  • background-position 背景图片的展开方式

    英文表示 :left//center//right(水平) top//center//bottom(垂直)

    数值表示 :2px(左间距) 3px(上间距)

    可以简写

    background:(背景颜色) (背景图片) (背景图片是否平铺) (附件 通常仅用在 body 标签) (水平 垂直)

    只有水平和垂直不能颠倒顺序

列表的属性

  • 去掉列表前面的项目符号 list-style-type:none; 可简写 list-style:none;
  • 用小图替换列表前的符号 list-style-image:url(图片的地址)；

表格合并边框线

  • 合并表格的边框线 border-collapse:collapse; (在html中<table rules="all">)

clear:left//right//both;

网页的布局思想

  1. 清除所有的html格式

  2. 页面属性设置 body{ }

    body{ font-size:14; font-family="宋体"; color=#000000; background-color:#e2e2e2; line-height:150%; }

    body{背景颜色/背景图片 字体大小 字体颜色 字体样式 行间距}

  3. 把整个页面划分盒子结构

行内元素和块元素

行内元素：输入完标签之后，不是独占一行。行内元素的宽度和高度由内容来决定，width height不能用。

块元素：输入完标签之后，自己独占一行。可以设置宽度和高度

块转转换行内 display:inline;

行内转换为块 display:block;

内容溢出

当内容溢出，如何显示 overflow

  • :hidden; 隐藏
  • :auto; 如果盒子装不下，就出现滚动条
  • :scroll; 无论是否装得下都有滚动条边框

继承

外层元素的样式，会被里面的元素所继承

可以继承的标签：所有文本属性

自己有的属性，不向外继承。

CSS选择器优先级

  • 单个选择器的优先级

    标签选择器 < 类选择器 < id 选择器 < 行内样式表

  • 复合选择器的优先级

    越具体的选择器优先级越高

    注：如果两个选择器完全相同，后出现的优先级高

  • 可以在任意属性值的后面加 !important 来强调这条规则比应用于同一元素的其他规则更重要

定位

position：定位方式有：

static 默认（不设置即默认） fixed 固定 relative 相对定位 absolute 绝对定位

坐标：偏离目标元素多远的距离

坐标的属性：left right top bottom

固定定位 fixed

  • 相对于浏览器窗口来进行的定位
  • 如果不设置定位坐标，就在原来的位置
  • 层级要比普通标签高
  • 如果结合定位坐标，就是相对于目标位置的距离

相对定位 relative

  • 相对定位占空间（自身原来的空间）
  • 如果不结合定位坐标，就是在原来的位置
  • 如果结合定位坐标，就是相对于自身原来的位置
  • 层级要高于普通的元素

绝对定位 absolute

  • 绝对定位不占空间

  • 层级高于普通元素

  • 不结合定位坐标，就是在原来的位置

DFS BFS

DFS思路

BFS和DFS选择

  1. BFS是用来搜索最短径路的解是比较合适的，比如求最少步数的解，最少交换次数的解，因为BFS搜索过程中遇到的解一定是离根最近的，所以遇到一个解，一定就是最优解，此时搜索算法可以终止。这个时候不适宜使用DFS，因为DFS搜索到的解不一定是离根最近的，只有全局搜索完毕，才能从所有解中找出离根的最近的解。（当然这个DFS的不足，可以使用迭代加深搜索ID-DFS去弥补）
  2. 空间优劣上，DFS是有优势的，DFS不需要保存搜索过程中的状态，而BFS在搜索过程中需要保存搜索过的状态，而且一般情况需要一个队列来记录。
  3. DFS适合搜索全部的解，因为要搜索全部的解，那么BFS搜索过程中，遇到离根最近的解，并没有什么用，也必须遍历完整棵搜索树，DFS搜索也会搜索全部，但是相比DFS不用记录过多信息，所以搜素全部解的问题，DFS显然更加合适。

shell常见命令

shell脚本内部可以执行两种命令：

  • 外部命令(external command): 可以在命令提示符中执行的普通命令
  • 内部命令(internal command): 内部命令是在shell内部实现的，不能作为外部程序被调用。（然而，大部分的内部命令同时也提供了独立运行的程序版本——属于POSIX规范）

常用命令：

  1. break 跳出循环

  2. 冒号(:)命令是一个空命令。

    它偶尔用于简化条件逻辑，相当于true的别名，while ：实现了无限循环。由于它是内置命令，所以运行的比true快，但是输出可读性差。

  3. continue 跳到下一次循环

  4. 点(.)命令，用于在当前shell中执行命令。

    ​ 通常，当一个脚本执行一条外部命令或脚本程序时，它会创建一个新的环境（一个子shell），命令将在这个行环境中执行，在命令执行完毕后，这个环境被丢弃，留下退出码返回给父shell。

    ​ 所以，当你要用一个脚本来为后续执行的命令设置环境时，这命令就非常有用，在这种意义上，点命令的作用类似于C/C++的#include指令。

    ​ 示例：假设有两个包含环境设置的文件，他们分别针对两个不同的开发环境。

    （1）为了设置老的环境，可以使用文件classic_set

    （2）对于新命令，使用文件lastest_set

    （3）可以通过将这些脚本程序和点命令结合来设置环境

  5. echo命令

    ​ 虽然，X/Open建议在现代shell中使用printf命令，但是使用echo方便。

    ​ 如何去掉换行符？

    • echo -n "str" -n参数去掉换行符

    • echo -e "str\c" -e参数确保启用了反斜线转义字符的解释(\c代表去掉换行符)

      老版本bash默认开启反斜线转义字符的解释，但最新版本的bash不一定开启。

    如果需要脚本兼容UNIX系统并且需要删除换行符，最好坚持使用printf命令。

    如果只运行在Linux和bash上，那么echo-n是不错的选择。

  7. exec命令 有两种用法,第二种略。

    典型用法是将当前shell替换为一个不同的程序。

    exec wall "Thanks for all the fish"

    脚本中的这个命令会用wall命令替换当前的shell。脚本程序中exec命令后面的代码都不会执行，因为执行这个脚本的shell已经不存在了。

  8. exit n 命令

    • 使脚本以退出码n结束运行。

    • 在交互式shell命令提示符中输入exit以退出系统。

    • 如果脚本程序在退出时不指定退出状态，那么脚本中最后一条被执行的命令状态将被用作返回值。

      0 表示成功

      128及以上 出现一个信号
